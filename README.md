# MyDiabby-Uploader-Linux
MyDiabby uploader for Linux.

## I am not affliated with MyDiabby or Tidepool.

## Install
- Download the zip file in the latest release.
- Extract the zip.
- Open a terminal in the 'Installer' folder of the extracted zip
- Run `sudo ./install.sh`

## Uninstall
- Download the uninstall.sh file in the latest release.
- Open a terminal
- Run `sudo ./uninstall.sh`
